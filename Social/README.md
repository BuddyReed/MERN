## Description

Socailize is a full stack MERN socail media application.

## Technologies Used

- React, Node.js, and Express
- HTML, CSS, Tailwind, and Material-UI
- Formik,Yup
- Redux Tollkit and Persistent
- React Dropzone
- Mongoose
- MongoDB
- JsonWebToken "JWT"
